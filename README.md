# SampLoop 🎧

SampLoop est une application mobile permettant de créer des ambiances sonores personnalisées à partir de vos propres enregistrements.

L'idée : assembler des sons comme un puzzle pour construire des paysages sonores uniques, même hors ligne.

---

## ✨ Fonctionnalités

- 🎵 Lecture simultanée de plusieurs pistes (jusqu'à 9)
- 🔁 Gestion de boucles (loops)
- 🎛️ Ajout d'effets sonores
- 🎨 Attribution de couleurs aux sons pour structurer visuellement les ambiances
- 📱 Fonctionne hors ligne (idéal pour enregistrement en extérieur)

---

## 🧠 Concept

Chaque son est une "pièce" que l’utilisateur assemble pour créer une ambiance globale.

SampLoop met l'accent sur :
- la simplicité d'utilisation
- la créativité sonore
- une approche visuelle du mix audio
